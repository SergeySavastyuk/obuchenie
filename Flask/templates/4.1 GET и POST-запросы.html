GET и POST-запросы

Создание таких полей возможно благодаря html. Для этого обязательно задается тег < form> с указанием метода POST или GET. Можно указать action, чтобы направить на другую страницу. Также необходим тег input с типом submit – эта специальная кнопка говорит сайту, что данные нужно обработать на сервере.

- action оставляем пустым и после нажатия кнопки отобразится этажа таблица

- label отображает надпись для Input с именем Сообщение:

- input будет однострочным полем для ввода, тип текст, имя message

- submit кнопка для отправки данных на сервер
