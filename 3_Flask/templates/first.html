Внутренние стили встраиваются в < head> и определяются внутри тега < style>. У них приоритет выше, чем у внешних, но уступают встроенным.
Селектор описывает элемент или группы элементов, к которым применяются CSS-свойства Селекторы класса позволяют задавать стили для одного и более элементов с одинаковым именем класса, размещенных в разных местах страницы. Пишется в теге < style> .my_class1{свойства со значениями} , а потом в нужный тег добавляется атрибут class="наименование вашего нужного класса"
Селектор идентификатора позволяет форматировать один конкретный элемент. Значение id должно быть уникальным, на одной странице может встречаться только один раз. Уникальный идентификатор элемента может использоваться как способ ссылки на конкретные части документа. При объявлении у такого селектора необходимо ставить знак # перед названием.
В шаблоне при помощи тега link производиться подключение. Атрибут rel указывает тип ссылки, значение stylesheet обозначает ссылку на таблицу стилей. В атрибуте href применяется особая функция: url_for, указывающая на название папки static и путь к самому файлу в аргументе filename с начальной точкой местположения от static.
Как подключить статические файлы, используется тег < script> далее указывается адрес через функцию url_for
